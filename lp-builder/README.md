# 🎨 LP Builder — 日本向け ランディングページ制作お助けアプリ

日本の EC サイトの LP（ランディングページ）を、**モジュールを組み合わせるだけで作れるビジュアルエディタ**です。Shopify を想定しつつ、画像・動画・文章を入れていくだけで完成します。

ブラウザだけで動き、ビルド不要。作成途中は右上のトグルで PC／タブレット／スマホ表示を逐次確認でき、完成したら **完全 HTML** と **Shopify Liquid section** の両方に出力できます。

**👉 [ブラウザで開く](https://mochamofu.github.io/GLM-test-app/lp-builder/)**

---

## ✨ 機能

- 🧩 **16 種の日本向け LP モジュール**（お知らせバー・ヒーロー・価格表・口コミ・FAQ…）
- 🎨 **各モジュール 3 デザインバリエーション**（計 48 パターン）。右ペインで切替え可能。切替時も入力内容は保持
- 🎨 **40 種のテンプレート**（商材ジャンル × 配色の組み合わせ）
  - コスメ・美容 14 種 ／ ファッション・雑貨 13 種 ／ デジタル・サービス 13 種
- 🖥️📱 **完全レスポンシブ**（プレビューも実出力も）
- 🔀 **ライブプレビュー**（右上トグルで PC 1280 / タブレット 768 / SP 375 を即時切替）
- ↶↷ **Undo / Redo**（Ctrl+Z / Ctrl+Y）
- 💾 **自動保存（localStorage）＋ JSON 入出力**（プロジェクトの持ち運び可）
- ⬇️ **2 種の出力**
  - **完全 HTML 1枚**（Shopify のページ本文にコピペで完成／単体でも動く）
  - **Shopify Liquid section**（Online Store 2.0 対応、テーマカラーを管理画面から変更可）

---

## 🚀 すぐ始める

`index.html` をブラウザで開くだけ。インストール不要です。

> 推奨: Chrome / Edge / Firefox の最新版

### 3ステップで LP 完成

1. **左ペイン「テンプレ」タブ** から、商材ジャンルと配色に合うテンプレを選ぶ
   - サンプル構成（10〜13 モジュール）と配色が一括で読み込まれます
2. **中央のプレビュー** でモジュールをクリック → **右ペイン** で画像・文章・色を編集
   - 右ペイン上部の **「デザイン」** で、そのモジュールの見た目バリエーション（3種）を切替えられます
3. **上部「HTML出力」/「Liquid出力」** ボタンで完成データを取得

これだけで日本向け LP が完成します。

## 🧩 16 種のモジュール

| # | モジュール | 何に使うか |
|---|---|---|
| 1 | 📢 お知らせバー | 画面最上部の細い告知帯（セール・送料無料） |

| 3 | 📝 リード文 | 導入文。見出し＋本文 |

| 5 | 🖼️ 画像＋文章（交互） | 画像と文章のペア、複数で左右交互レイアウト |
| 6 | 📋 ステップ・手順 | 使い方・流れを順番に（漢数字付き） |
| 7 | ⚖️ 比較表 | 他社/従来品との比較、自社列だけハイライト |

| 10 | 🖼️ ギャラリー | 商品画像の並列表示（2〜4カラム） |
| 11 | ⭐ お客様の声 | 口コミと星評価、平均評価表示付き |
| 12 | ❓ よくある質問 | Q&Aの折りたたみ（アコーディオン） |
| 13 | 🎬 動画 | YouTube / MP4 埋込 |
| 14 | ⏰ カウントダウンタイマー | セール終了までの残り時間をリアルタイム表示 |

## ⬇️ 出力（エクスポート）

### A. 完全 HTML（おすすめ：手軽）

画面上部の **「⬇️ HTML出力」** ボタン。

- CSS を1枚に内包した完全 standalone HTML
- **使い方**: Shopify 管理画面 → **オンラインストア** → **ページ** → 新規ページ → HTML表示 をオン → 本文に貼り付け
- 単体の `.html` ファイルとしても動くので、テスト用にも便利

### B. Shopify Liquid section（テーマ開発者向け）

画面上部の **「⬇️ Liquid出力」** ボタン。

- Online Store 2.0 形式の section ファイル
- **使い方**:
  1. Shopify 管理画面 → **オンラインストア** → **テーマ** → **コードを編集**
  2. `sections/` フォルダに `lp-builder.liquid` を新規作成して貼り付け
  3. テーマエディタの「セクションを追加」→「LP Builder」を選択
- テーマカラー等は section settings から動的に変更可能

## 💾 保存・読込

- **自動保存**: 編集内容はブラウザの localStorage に自動保存（同一ブラウザで再開可）
- **JSON 保存**: **「💾 保存」** ボタンでプロジェクトファイル（`.json`）をダウンロード
- **JSON 読込**: **「📂 読込」** ボタンからプロジェクトファイルを復元（別端末への持ち運びに便利）

## 📁 ファイル構成

```
lp-builder/
├── index.html              ← エディタUI（3ペイン）
├── README.md               ← このファイル
├── build-styles.js         ← CSS→JS 変換スクリプト
├── css/
│   ├── editor.css          ← エディタ画面のスタイル
│   └── preview.css         ← LP 本体スタイル（編集元）
└── js/
    ├── lp-styles.js        ← preview.css を文字列化（自動生成）
    ├── templates.js        ← 40テンプレート定義
    ├── modules.js          ← 16モジュール定義（スキーマ＋レンダラ）
    ├── persistence.js      ← localStorage ＋ JSON 入出力
    ├── state.js            ← プロジェクト状態 ＋ Undo/Redo
    ├── renderer.js         ← 状態→HTML ドキュメント
    ├── inspector.js        ← 編集フォーム自動生成
    ├── preview.js          ← プレビュー iframe ＋ デバイストグル
    ├── export.js           ← HTML/Liquid 出力
    └── app.js              ← 起動・イベント結線
```

### CSS を編集したい場合

`css/preview.css` を編集後、以下を実行して `js/lp-styles.js` を再生成してください（CSS が文字列として JS に埋め込まれることで、`file://` でもエクスポート HTML が単体動作します）。

```bash
node build-styles.js
```

## ⚙️ 技術方針

- **バニラ HTML/CSS/JS**（フレームワークなし・ビルド不要）
- **`file://` で動く**: プレビューもエクスポートも、ローカルファイル単体で動作
- CSS は `js/lp-styles.js` に文字列埋め込み（`file://` での fetch 制限を回避）
- プレビューは iframe ＋ `srcdoc`、デバイストグルで幅を切替
- 履歴は JSON スナップショット方式（上限60件）
